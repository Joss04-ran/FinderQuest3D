# FinderQuest3D

A 3D Game made in Windows Form using SharpDX 

The concept : 
A 3D adventure puzzle game built using C# Windows Forms and the SharpDX library. The player explores different 3D environments, interacts with various NPCs, and tests their knowledge by answering a series of questions across different subjects.

Gameplay : 
- **Movement:** Use `W`, `A`, `S`, `D` keys to move around the 3D map.
- **Camera:** Use the `Left` and `Right` arrow keys to rotate the camera.
- **Interaction:** Press `Enter` to interact with an NPC when near them.
- **Answering:** Press `Y` to bring up the question prompt when interacting with an NPC.
- **Objective:** Answer the questions given by NPCs correctly to progress. Complete all tasks in an area to unlock the transition to the next map level before the time runs out. The game features Easy and Hard modes, affecting question difficulty and time limits.
